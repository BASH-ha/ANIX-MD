## 📌 About

**ANIX-MD** is a powerful WhatsApp bot that helps you manage groups, automate tasks, and have fun with various commands. Built with Baileys library and optimized for performance.

## 📱 Local Setup

### Prerequisites
- Node.js 18+ installed
- WhatsApp account

### Installation

```bash
git clone https://github.com/BASH-ha/ANIX-MD.git
cd ANIX-MD
npm install
echo "SESSION_ID=ANIX-MD:~YOUR_SESSION_ID" > .env
npm start
